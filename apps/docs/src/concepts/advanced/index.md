# Advanced

Advanced pages cover protocol, deployment, and ceremony details for readers who
already understand the main model.

Introductory pages should explain what each boundary protects and when an
operation happens. Advanced pages should explain transcripts, proofs, deployment
roles, route auth, parity checks, rollback fencing, and ceremony-level failure
handling.
