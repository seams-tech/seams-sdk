# Seams v9 Imagegen Gradients

Source: built-in imagegen, generated from the Seams v9 gradient logo references and refined against the provided update-card screenshot as the visual style reference.

Output:
- `seams-v9-imagegen-gradient-sheet.png`: original 3x3 imagegen sheet.
- `seams-v9-gradient-*-1024.png`: nine square crops with soft blurred gradient fields and fine grain.
- `seams-v9-gradient-*-2048.png`: higher-resolution exports upscaled from the 1024 crops with Lanczos resampling and light sharpening.
