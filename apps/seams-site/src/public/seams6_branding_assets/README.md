# Seams Branding Asset Package

This package contains a monochrome Seams app icon system with black lines and dots on a full-bleed warm cream background.

## Included

- `master/` — full-bleed square PNG masters at 2048, 1024, and 512 px.
- `ios/AppIcon.appiconset/` — iOS-ready AppIcon asset catalog with `Contents.json` and required iPhone/iPad/App Store icon sizes.
- `web/` — favicon and web/app icon exports from 16 px through 1024 px, plus `favicon.ico`.
- `marketing/` — common square export sizes for presentations, landing pages, and social use.
- `transparent_mark/` — black monogram-only PNGs with transparent background.
- `preview/` — a rounded icon preview and an asset sheet.

## Core colors

- Seams Black: `#1E1E1E`
- Seams Cream: `#FCF2E4`

## iOS note

Use the square, full-bleed images in `AppIcon.appiconset` for iOS. Do not manually round the production app icons; iOS applies the rounding mask automatically. The rounded files in `preview/` are for visual review only.
