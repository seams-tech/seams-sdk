# Test Review

## Rubric

- **Core functionality**: checks real runtime/business behavior and would fail on meaningful regressions.
- **Guardrail (non-tautological)**: structural/API invariants (imports/exports/no-legacy/boundaries). Useful, but not feature-flow coverage.
- **Shallow sanity check**: valid smoke checks with limited defect-detection depth.
- **Mixed (core + shallow)**: mostly core, with at least one shallow assertion.
